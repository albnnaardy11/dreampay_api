# Introduction

Welcome to the **DreamPay API Documentation**.

DreamPay is a robust, high-integrity financial backend specially designed for the **Santri Market Day** ecosystem. It provides a secure platform for santri to manage their funds, perform P2P transfers, and participate in a cashless marketplace.

## Key Principles
- **Speed**: Optimized for real-time transactions during busy market events.
- **Security**: Deterministic PIN hashing and strict RBAC.
- **Integrity**: Transactional safety ensuring zero-balance anomalies.

## Getting Started
To use the API, you must first register an account. By default, new accounts are registered with the **Santri** role. 

Base URL: `http://localhost/api` (or your production URL)
